# Catering Food Quantity Estimator

## Description

This project estimates food quantities for catering events based on:

- Number of guests
- Menu items
- Serving sizes
- Food categories
- 10% buffer for extra servings

## Requirements

- Python 3.10 or later

# Running the Program

From the project directory, run:

```bash
python food_quantities.py
```

The program will execute the sample menu and display the estimated food quantities.

---

# Running the Automated Tests

Run all automated tests with:

```bash
pytest
```
Run:

```bash
python test_food_quantities.py
```
Runtime Complexity:

```
O(n)
```

Memory Complexity:

```
O(n)
```

where **n** is the number of menu items.
